# Perception Module

Este módulo maneja toda la percepción visual en tiempo real.

## Componentes (próximamente en PHASE 1+)

- `camera.py`: Captura desde webcam/ESP32-CAM
- `detector.py`: Detección de objetos (YOLO/MobileNet)
- `visualizer.py`: Visualización de bounding boxes
- `recorder.py`: Grabación de detecciones
